# AI Career Guidance Website

## Overview

The AI Career Guidance Website is a web application that helps users discover suitable career paths based on their interests, skills, and work preferences.

The application uses Artificial Intelligence to analyze user preferences and generate personalized career suggestions. The backend is built with Node.js and Express, while the frontend is developed using HTML, CSS, and JavaScript.

The AI responses are generated using the Groq API with the Llama 3.3 70B Versatile model.

---

## Features

- AI-powered career recommendations
- User-friendly interface

## Installation

### Step 1

Clone the repository

```bash
git clone https://github.com/aryansonune/careerguidance-backend.git
```

### Step 2

Open the project folder

```bash
cd YOUR_REPOSITORY
```

### Step 3

Install dependencies

```bash
npm install
```

---

## Environment Variables

Create a `.env` file in the root folder.

Example:

```env
GROQ_API_KEY=your_groq_api_key
PORT=3000
```

---

## Running the Project

Start the server

```bash
npm start
```

Server will run on

```
http://localhost:3000
```

---

## API Endpoint

### POST

```
/api/suggestions
```

### Request Body

```json
{
    "preferences":"Artificial Intelligence, Programming, Remote Work"
}
```

### Response

```json
{
    "modelUsed":"llama-3.3-70b-versatile",
    "suggestions":"Career suggestions generated by AI..."
}
```
